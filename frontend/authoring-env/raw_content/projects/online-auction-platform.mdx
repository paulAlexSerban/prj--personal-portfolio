## Description:
An online auction platform allows users to bid on and purchase items in a competitive and interactive environment. This project showcases your ability to create a visually appealing and interactive front-end application, integrating various components to enhance user experience.

## Features:

- Data Grid: Display a list of auction items, with relevant details such as item name, category, current bid, and time remaining. Allow sorting, filtering, and pagination.
- Charts: Visualize auction data using various charts, including bar charts for bidding activity, line charts for price trends, and pie charts for item category distribution.
- Scheduler: Allow users to schedule reminders for upcoming auctions, bid deadlines, and payment due dates.
- Date Picker: Enable users to select a date range for filtering auction items and scheduling reminders.
- Rich Content Editor: Provide a platform for creating and editing item descriptions, auction terms, and user reviews, with formatting options, images, and tables.
