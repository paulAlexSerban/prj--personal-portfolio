## Description:

An employee performance management system allows managers to track employee performance, set goals, and provide feedback. This project showcases your ability to create a visually appealing and interactive front-end application that effectively manages employee data.

## Features:

-   Data Grid: Display a list of employees, with relevant details such as job title, department, performance rating, and goals. Allow sorting, filtering, and pagination.
-   Charts: Visualize employee performance data using various charts, including bar charts for goal completion, line charts for performance trends, and pie charts for department distribution.
-   Scheduler: Allow managers to schedule performance reviews, goal-setting meetings, and reminders related to employee development.
-   Date Picker: Enable users to select a date range for filtering employee performance data and scheduling events.
-   Rich Content Editor: Provide a platform for creating and editing performance reviews, goal descriptions, and feedback notes, with formatting options, images, and tables.
