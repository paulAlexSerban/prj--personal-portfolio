- `mount` - displays a list of media devices currently mounted on the system
  - `mount -t ext4` - displays all filesystems with the type of `ext4`
  - `mount -t vfat /dev/sdb1 /media/disk` - mount the USB memory stick at device `/dev/sdb1` at location `/media/disk`
  - `-o` - this option allows you to mount the filesystem with a comma-separated
    - `check=none` - mount the filesystem without performing an integrity check
- `umount [directory | device]` - use to unmount a media device from the system
  - `umount /dev/sdb1` - unmount the USB memory stick at device `/dev/sdb1`
  - `umount /media/disk` - unmount the USB memory stick at location `/media/disk`
- `fdisk` - use to create, delete, and modify partitions on a hard disk
  - `fdisk -l` - list all the partitions on the system
  - `fdisk /dev/sdb` - use to create, delete, and modify partitions on the USB memory stick at device `/dev/sdb`
- `mkfs` - use to create a filesystem on a partition
  - `mkfs.ext4 /dev/sdb1` - create an `ext4` filesystem on the USB memory stick at device `/dev/sdb1`
  - `mkfs.vfat /dev/sdb1` - create a `vfat` filesystem on the USB memory stick at device `/dev/sdb1`
- `mkswap` - use to create a swap partition
  - `mkswap /dev/sdb1` - create a swap partition on the USB memory stick at device `/dev/sdb1`
- `swapon` - use to activate a swap partition
  - `swapon /dev/sdb1` - activate the swap partition on the USB memory stick at device `/dev/sdb1`
