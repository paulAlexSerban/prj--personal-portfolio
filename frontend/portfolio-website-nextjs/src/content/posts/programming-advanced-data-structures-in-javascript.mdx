## Trees
A tree is a hierarchical data structure that consists of nodes connected by edges. Each node can have multiple child nodes, except for the root node, which has no parent node. In JavaScript, we can implement a tree using objects and references. For example:

```javascript
class TreeNode {
  constructor(value) {
    this.value = value;
    this.children = [];
  }

  addChild(childNode) {
    this.children.push(childNode);
  }
}

const rootNode = new TreeNode(1);
const childNode1 = new TreeNode(2);
const childNode2 = new TreeNode(3);

rootNode.addChild(childNode1);
rootNode.addChild(childNode2);

console.log(rootNode); // Output: TreeNode { value: 1, children: [TreeNode { value: 2, children: [] }, TreeNode { value: 3, children: [] }]}
```

In the above example, we have implemented a tree using the TreeNode class. We can add child nodes to the tree using the addChild method.

## Graphs
A graph is a non-linear data structure that consists of nodes connected by edges. Unlike trees, graphs can have cycles and do not have a root node. In JavaScript, we can implement a graph using objects and references. For example:

```javascript
class GraphNode {
  constructor(value) {
    this.value = value;
    this.neighbors = [];
  }

  addNeighbor(neighborNode) {
    this.neighbors.push(neighborNode);
  }
}

const node1 = new GraphNode(1);
const node2 = new GraphNode(2);
const node3 = new GraphNode(3);

node1.addNeighbor(node2);
node2.addNeighbor(node3);
node3.addNeighbor(node1);

console.log(node1); // Output: GraphNode { value: 1, neighbors: [GraphNode { value: 2, neighbors: [GraphNode { value: 3, neighbors: [GraphNode { value: 1, neighbors: [Circular] }] }] }] }
```
In the above example, we have implemented a graph using the GraphNode class. We can add neighbors to a node using the addNeighbor method.

## Hash Tables
A hash table is a data structure that uses a hash function to map keys to values. Hash tables provide constant-time complexity for insertion, deletion, and retrieval operations. In JavaScript, we can implement a hash table using objects. For example:

```javascript
class HashTable {
  constructor() {
    this.table = {};
  }

  hash(key) {
    let hash = 0;
    for (let i = 0; i < key.length; i++) {
      hash += key.charCodeAt(i);
    }
    return hash;
  }

  set(key, value) {
    const hashedKey = this.hash(key);
    this.table[hashedKey] = value;
  }

  get(key) {
    const hashedKey = this.hash(key);
    return this.table[hashedKey];
  }

  delete(key) {
    const hashedKey = this.hash(key);
    delete this.table[hashedKey];
  }
}

const myHashTable = new HashTable();

myHashTable.set("one", 1);
myHashTable.set("two", 2);
myHashTable.set("three", 3);

console.log(myHashTable.get("two")); // Output: 2

myHashTable.delete("two");

console.log(myHashTable.get("two"));
// Output: undefined
```
In the above example, we have implemented a hash table using the HashTable class. We can set, get, and delete key-value pairs from the hash table using the set, get, and delete methods, respectively.
